# vibe-admin 문서

이 폴더는 vibe-admin 프로젝트의 **기획·구현·보안·테스트·운영** 문서를 역할별로 나눠 정리한 곳입니다.
초기 한 덩어리 프롬프트로 만들기 시작했지만, 실제 구현은 여러 차례 바뀌었습니다(인증 방식, 예약 UX 등).
이 문서들은 **현재 코드 기준**으로 다시 정리한 결과입니다.

## 빠른 요약

- **무엇**: 소규모 고객관리 어드민 (예약 + 문의 + 고객 메모)
- **스택**: Next.js(App Router) + Supabase(Auth/Postgres/RLS) + Vercel
- **인증**: 이메일/비밀번호 (회원가입 시 이메일 인증 1회)
- **예약**: 네이버식 캘린더 + 연속 시간 슬롯 다중 선택, 중복 방지, 고객 취소
- **공개 스케줄**: 월간 캘린더(승인 예약만, 개인정보 미노출)
- **핵심 원칙**: 클라이언트 입력 불신 + RLS + 개인정보 최소 노출
